\chapter{致谢}

	修读双学位的经历是我人生中最重要的一笔财富之一。在过去的本科生涯中，从东校区跑到南校区，不辞劳苦，虔诚地到一教或者艺术楼上课，享受每次汲取知识的喜悦。在修读双学位的过程中，有一群志同道合的朋友们，可以和他们讨论数学，这也是我每次到南校最开心的事。

	最后感谢张俊玉老师对本篇论文的耐心指导，也感谢在书写这篇论文的过程中帮助到我的朋友们。
